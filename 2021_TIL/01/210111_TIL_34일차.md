### CSS
 * 타이포그래피
  * 글꼴 크기 변경 속성 : Px, rem ... 
   * 현재는 rem을 선호
  * 색깔 변경 속성
   * color에 16진수, rgb(숫자,숫자,숫자), 실제 색 이름 을 부여해서 색상 변경
  * 텍스트 정렬
   * text-align에 left, center, right, justify 등을 이용해서 정렬
  * font 관련 제어
   * font-family // font 글꼴 지정
   * font-weight // font 두께 지정
   * line-height // 줄 간격 지정(기본 값 : 1.2)
