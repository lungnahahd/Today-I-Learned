### 알고리즘 풀이
  * Python 3를 활용한 stack 구현
    * push, pop, top, empty, size 구현
    * 리스트 선언 재학습
    ```
    list = [] // 길이가 없는 리스트 선언
    list = [0 for i in range(n)] // n 크기를 갖는 리스트 선언
    
    list = [[0 for col in range(n)] for row in range(n)] // 0으로 초기화 되어있는 2차원 리스트 선언
    ```
