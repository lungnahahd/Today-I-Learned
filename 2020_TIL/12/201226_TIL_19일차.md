### Backend
  #### Git 관리
   * VS code에서 git 관리 학습
      * git remote set-url 명령어를 통해 터미널을 이용한 원격 저장소 변경 가능
      * git checkout -t 명령어를 통해 터미널을 이용한 원격 저장소 branch 선택 가능
      * clone을 활용하여 팀원과 협업 학습
        * 현재 진행하고 있는 팀프로젝트에서 팀원과 관련 코드를 clone하고 여기서 branch를 이용하여 협업 진행 중...
