### Python
> #### 람다식
>   * 람다식으로 메모리 절약, 가독성 향상, 코드 간결화를 얻을 수 있음
>   * 함수와의 차이점 존재
>     * 함수 = 객체 생성 -> 리소스(메모리) 할당
>     * 람다식 = 즉시 실행 (Heap 초기화, 메모리 초기화)
>     * 또한 람다식을 함수의 parameter로 사용할 수 도 있음
>       * 이 사용으로 람다식이 드디어 크나 큰 장점을 보이게 됨
